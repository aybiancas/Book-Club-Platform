# Book Club Platform

#### A simple Java social app for book club members and managers to handle a database of books and create reading lists, posts, comments and schedule meetings.


Members will be able to:
- handle reading lists
- handle their club reading progress
- create posts
- add comments to posts
- attend meetings
- search for books in the club database

Managers will be able to:
- add and remove users
- add and remove books
- schedule club meetings
- remove meetings from schedule
